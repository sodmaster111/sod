# Архитектура SODMASTER

## Frontend
Веб-интерфейс позволяет управлять агентами, запускать сценарии и отслеживать результаты в режиме реального времени. Он отвечает за удобное взаимодействие пользователя с системой и визуализацию данных.

## Backend
Серверная часть координирует работу агентов, управляет очередями задач и обеспечивает интеграцию с внешними сервисами. Здесь реализована бизнес-логика, маршрутизация запросов и хранение состояния.

## Ollama
Ollama используется как источник языковых моделей для генерации текстов и анализа данных. Интеграция обеспечивает гибкую настройку моделей и возможность локального исполнения запросов для сохранения приватности.

## Telegram
Бот в Telegram предоставляет быстрый доступ к функциям SODMASTER через чат: запуск задач, получение отчетов и уведомлений. Это облегчает взаимодействие с платформой без необходимости заходить в веб-интерфейс.
